Morse Code Translator (8086 Assembly)
  A Morse Code Translator implemented in 8086 Assembly Language as part of the CSE 341: Microprocessors course project of BRAC University.
  Its a group project done by Swapnil Rob,Nafisa Hasan and Ahad Islam Shanto.

Features
  Encode English text (A–Z, 0–9, .,?!) into Morse Code.
  Decode Morse Code back into English text.
  Display Morse Code lookup table.
  Play Morse Code sounds (beeps for dot/dash).
  Adjustable speed (Fast, Medium, Slow).
  Interactive menu system with input validation.

Usage
  Open the code in EMU8086.
  Assemble and run.
  Follow the on-screen menu prompts.
