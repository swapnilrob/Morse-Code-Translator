Morse Code Translator (8086 Assembly):
  A Morse Code Translator implemented in 8086 Assembly Language as part of the CSE 341: Microprocessors course project of BRAC University.
  Its a group project done by Swapnil Rob,Nafisa Hasan and Ahad Islam Shanto.

Features:
  1.Encode English text (A–Z, 0–9, .,?!) into Morse Code.
  2.Decode Morse Code back into English text.
  3.Display Morse Code lookup table.
  4.Play Morse Code sounds (beeps for dot/dash).
  5.Adjustable speed (Fast, Medium, Slow).
  6.Interactive menu system with input validation.

Usage:
  1.Open the code in EMU8086.
  2.Assemble and run.
  3.Follow the on-screen menu prompts.
